## 1. 회원가입하기

### 📍 1단계: 웹사이트 접속

1. 인터넷 주소창에 입력: `https://marketing-automation-pink.vercel.app`
2. 오른쪽 위 **"시작하기"** 버튼 클릭

### 📍 2단계: 회원가입 정보 입력

**필요한 정보:**
- 📧 이메일 주소 (예: hong@gmail.com)
- 🔒 비밀번호 (6자 이상, 영문+숫자 조합)

**입력 예시:**
```
이메일: myemail@gmail.com
비밀번호: mypass123!
비밀번호 확인: mypass123!
```

### 📍 3단계: 이메일 인증

1. 입력한 이메일로 **인증 메일**이 옵니다
2. 메일함에서 인증 링크를 클릭하세요
3. "인증 완료!" 메시지가 나오면 성공! ✅

## 2. 로그인하기

### 📍 로그인 방법

1. 홈페이지 오른쪽 위 **"로그인"** 클릭
2. 회원가입할 때 사용한 **이메일**과 **비밀번호** 입력
3. **"로그인"** 버튼 클릭

**💡 팁:** "로그인 상태 유지"를 체크하면 다음에 자동으로 로그인돼요!

---

## 3. 브랜드 등록하기

처음 로그인하면 **브랜드(제품/서비스) 정보**를 먼저 등록해야 해요.
AI가 여러분의 브랜드에 맞는 콘텐츠를 만들기 위해 필요합니다.

### 📍 1단계: 브랜드 설정 페이지 이동

1. 왼쪽 메뉴에서 **"브랜드"** 클릭
2. **"새 브랜드 추가"** 버튼 클릭

### 📍 2단계: 브랜드 정보 입력

**필수 정보:**

| 항목 | 설명 | 예시 |
|------|------|------|
| 브랜드 이름 | 제품/회사 이름 | "김밥천국 강남점" |
| 설명 | 무엇을 파는지 간단히 | "저렴하고 맛있는 김밥집" |
| 제품 종류 | 어떤 업종인지 | "음식점" |
| 타겟 고객 | 누구에게 팔건지 | "20-30대 직장인" |

**입력 예시:**
```
브랜드 이름: 김밥천국 강남점
설명: 저렴하고 푸짐한 김밥을 파는 동네 맛집
제품 종류: 음식점 (분식)
타겟 고객: 20-30대 직장인, 학생
```

### 📍 3단계: 저장하기

- 모든 정보를 입력했으면 **"저장"** 버튼 클릭
- "브랜드가 등록되었습니다!" 메시지 확인 ✅

## 4. 콘텐츠 만들기

이제 본격적으로 AI가 콘텐츠를 만들어줍니다!

### 📍 1단계: 콘텐츠 생성 페이지 이동

1. 왼쪽 메뉴에서 **"콘텐츠 생성"** 클릭
2. 콘텐츠 만들기 화면이 나타남

### 📍 2단계: 설정 선택하기

#### A. 콘텐츠 종류 선택

- **📝 텍스트**: SNS 게시물, 블로그 글
- **🎬 비디오**: 영상 스크립트

**초보자는 "텍스트"를 추천해요!**

#### B. 브랜드 선택

- 아까 등록한 브랜드를 선택하세요
- 예: "김밥천국 강남점"

#### C. 주제 입력

무엇에 대한 글을 쓸 건지 입력하세요.

**좋은 주제 예시:**
- ❌ 나쁜 예: "김밥" (너무 짧음)
- ✅ 좋은 예: "점심시간 직장인을 위한 김밥 할인 이벤트"

**주제 입력 팁:**
```
구체적일수록 좋아요!
"누구에게" + "무엇을" + "왜" 형식으로 써보세요.

예시:
- 직장인을 위한 점심 특가 메뉴 소개
- 다이어트 중인 학생에게 추천하는 건강 김밥
- 비오는 날 생각나는 따뜻한 라면 이벤트
```

#### D. 플랫폼 선택

어디에 올릴 건지 선택하세요:

- **Thread**: 한국 트위터 같은 SNS
- **LinkedIn**: 비즈니스용 SNS
- **Instagram**: 사진 중심 SNS
- **Naver Blog**: 네이버 블로그

**처음에는 "Thread"를 추천해요!**

#### E. 톤 선택 (말투)

어떤 느낌으로 쓸지 선택:

- **전문적**: 격식있는 말투 (예: "고객님께 알려드립니다")
- **친근한**: 친구 같은 말투 (예: "오늘 점심은 김밥 어때요?")
- **캐주얼**: 편한 말투 (예: "요즘 대박 핫한 김밥!")

**음식점이면 "친근한"을 추천!**

#### F. 길이 선택

- **짧게**: 1-2문단 (SNS 게시물)
- **보통**: 3-5문단 (블로그 글)
- **길게**: 5문단 이상 (자세한 설명)

**처음에는 "보통"을 추천!**

### 📍 3단계: AI가 만들기 시작!

모든 설정을 다 했으면:

1. **"AI 콘텐츠 생성"** 버튼 클릭
2. 10-30초 기다리면...
3. 오른쪽 화면에 **완성된 글**이 나타납니다! 🎉

### 📍 4단계: 결과 확인하고 수정하기

**만들어진 글 확인:**
- 오른쪽 화면에서 AI가 만든 글을 읽어보세요
- 마음에 들지 않으면 **다시 생성** 가능해요

**수정이 필요하면:**
- 주제를 더 구체적으로 바꿔보세요
- 톤이나 길이를 다르게 선택해보세요

### 📍 5단계: 저장하기

마음에 드는 글이 나왔으면:

1. **"저장"** 버튼 클릭
2. "저장되었습니다!" 메시지 확인
3. 언제든지 **"콘텐츠"** 메뉴에서 다시 볼 수 있어요!

## 5. 만든 콘텐츠 확인하기

### 📍 콘텐츠 목록 보기

1. 왼쪽 메뉴에서 **"콘텐츠"** 클릭
2. 내가 만든 모든 콘텐츠가 리스트로 나타남

**목록에서 할 수 있는 것:**
- 📄 콘텐츠 내용 다시 보기
- ✏️ 수정하기
- 📋 복사해서 SNS에 붙여넣기
- 🗑️ 삭제하기

### 📍 콘텐츠 복사해서 사용하기

1. 원하는 콘텐츠 클릭
2. 글 전체를 마우스로 드래그
3. **Ctrl+C** (맥은 Cmd+C) 눌러서 복사
4. Thread, Instagram 등에 가서 **Ctrl+V** (맥은 Cmd+V) 붙여넣기
5. 게시! 🚀

## 6. 자주 묻는 질문

### Q1. 무료인가요?

**A.** 기본 기능은 무료로 사용할 수 있어요.
더 많은 기능이 필요하면 유료 플랜도 있습니다.

---

### Q2. 하루에 몇 개까지 만들 수 있나요?

**A.** 무료 사용자는 하루 10개까지 만들 수 있어요.
유료 플랜은 무제한입니다!

---

### Q3. AI가 만든 글을 수정할 수 있나요?

**A.** 네! 복사한 뒤 자유롭게 수정해서 사용하세요.
AI가 만든 글은 "초안"이라고 생각하시면 됩니다.

---

### Q4. 어떤 SNS에 사용할 수 있나요?

**A.** Thread, LinkedIn, Instagram, Naver 블로그 등
어디든 복사해서 붙여넣기만 하면 됩니다!

---

### Q5. 영어로도 만들 수 있나요?

**A.** 네! 언어 설정을 바꾸면 영어 콘텐츠도 만들 수 있어요.
오른쪽 위 언어 전환 버튼을 클릭하세요.

---

### Q6. 콘텐츠가 마음에 안 들어요. 어떻게 하죠?

**A.** 다시 만들어보세요!

**더 좋은 결과를 위한 팁:**
1. **주제를 더 구체적으로** 써보세요
   - ❌ "김밥"
   - ✅ "점심시간 직장인을 위한 김밥 세트 할인 이벤트"

2. **톤을 바꿔보세요**
   - 전문적 → 친근한 으로 변경

3. **브랜드 설명을 더 자세히** 작성하세요
   - 무엇이 특별한지, 어떤 장점이 있는지

---

### Q7. 비디오는 어떻게 만드나요?

**A.** 콘텐츠 종류에서 "비디오"를 선택하면
영상 스크립트와 간단한 비디오를 만들 수 있어요.

**비디오 만들기 과정:**
1. 콘텐츠 종류: **"비디오"** 선택
2. 주제 입력 (예: "신메뉴 소개 영상")
3. AI 모델은 **"Claude"** 추천
4. 생성 버튼 클릭!
5. 30초~1분 기다리면 완성

⚠️ **참고:** Ollama 모델은 현재 사용할 수 없어요!

---

### Q8. 저장한 콘텐츠를 삭제하려면?

**A.**
1. 콘텐츠 목록 페이지로 이동
2. 삭제할 콘텐츠의 **🗑️ 삭제** 버튼 클릭
3. "정말 삭제하시겠습니까?" 확인 → **"예"**

---

### Q9. 여러 개 브랜드를 등록할 수 있나요?

**A.** 네! 여러 브랜드/제품을 등록하고 전환할 수 있어요.
- 브랜드 페이지에서 **"새 브랜드 추가"**
- 콘텐츠 생성 시 원하는 브랜드 선택

---

### Q10. 모바일에서도 사용할 수 있나요?

**A.** 네! 스마트폰 브라우저에서도 사용 가능해요.
같은 주소로 접속하시면 됩니다.

## 💡 성공적인 콘텐츠 만들기 팁

### 팁 1: 구체적인 주제 쓰기

**나쁜 예시:**
- "김밥"
- "할인"
- "신메뉴"

**좋은 예시:**
- "점심시간 직장인을 위한 김밥 3종 세트 20% 할인"
- "다이어트 중이라면? 저칼로리 야채 김밥 출시"
- "비오는 날 따뜻한 떡볶이 무료 사이드 이벤트"

### 팁 2: 브랜드 정보 자세히 쓰기

브랜드를 등록할 때 특징을 많이 적을수록 좋아요!

**예시:**
```
설명:
강남역 5번 출구 앞 20년 전통 김밥집.
신선한 재료만 사용하고, 모든 김밥을 주문 즉시 만듭니다.
특히 참치김밥과 치즈김밥이 인기 메뉴이며,
점심시간에는 직장인 고객이 많이 찾아옵니다.
```

### 팁 3: 여러 번 생성해보기

AI가 만든 첫 번째 결과가 마음에 안 들면
**주제를 살짝 바꿔서** 다시 생성해보세요!

같은 주제로 3-4번 만들어보면 그 중 하나는 마음에 들 거예요.

### 팁 4: AI 결과를 그대로 쓰지 말고 수정하기

AI가 만든 글을 **나만의 스타일**로 조금 수정하세요.

**예시:**
- AI가 만든 글: "고객님께 특별한 할인을 제공합니다"
- 내 스타일로 수정: "여러분께 깜짝 할인 이벤트! 🎉"
