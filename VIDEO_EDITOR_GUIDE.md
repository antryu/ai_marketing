# 🎬 비디오 편집기 사용 가이드

마케팅 자동화 플랫폼에 통합된 AI 기반 비디오 편집기입니다.

> **IMPORTANT: Language Rules**
> - **NEVER use Chinese (Hanzi), Japanese (Kanji/Hiragana/Katakana), or any CJK characters**
> - Allowed languages: **Korean (Hangul) and English ONLY**
> - This rule applies to ALL files in the project

## ✨ 주요 기능

### 1. **AI 비디오 자동 생성**
- 주제만 입력하면 AI가 자동으로 비디오 스크립트 생성
- 플랫폼별 최적화 (YouTube, Instagram, TikTok, LinkedIn)
- 브랜드 정보 기반 맞춤형 콘텐츠

### 2. **직관적인 비주얼 편집기**
- 실시간 캔버스 프리뷰
- 드래그 앤 드롭 씬 관리
- 타임라인 기반 편집

### 3. **전문가 디자인 템플릿**
- 제품 소개
- 프로모션
- 스토리텔링
- 고객 후기

### 4. **강력한 편집 도구**
- 텍스트, 이미지, 도형 추가
- 그라데이션 & 컬러 배경
- 애니메이션 효과
- 씬 전환 효과

## 🚀 사용 방법

### 1단계: 콘텐츠 유형 선택

콘텐츠 생성 페이지에서 **비디오** 옵션을 선택합니다.

```
콘텐츠 생성 → 콘텐츠 유형: [비디오] 선택
```

### 2단계: 기본 정보 입력

- **제품**: 비디오를 만들 제품/브랜드 선택
- **토픽**: 비디오 주제 입력 (예: "신제품 출시 안내")
- **플랫폼**: YouTube, Instagram, TikTok, LinkedIn 중 선택
- **톤**: 전문적, 캐주얼, 친근함 등

### 3단계: AI 비디오 생성

"AI 비디오 생성" 버튼을 클릭하면:

1. **AI 분석**: 브랜드 정보와 주제를 분석
2. **스크립트 생성**: 플랫폼에 최적화된 스크립트 작성
3. **씬 구성**: 자동으로 씬 분할 및 타이밍 설정
4. **비주얼 디자인**: 배경, 텍스트, 색상 자동 설정

### 4단계: 비디오 편집기

AI 생성이 완료되면 비디오 편집기가 열립니다.

#### 편집기 레이아웃

```
┌─────────────────────────────────────────────────────┐
│ 헤더: 프로젝트 이름, 저장, 내보내기                │
├──────┬─────────────────────────────┬────────────────┤
│      │                             │                │
│ 도구 │     캔버스 (프리뷰)         │  씬 편집 패널  │
│ 바   │                             │                │
│      │                             │                │
├──────┴─────────────────────────────┴────────────────┤
│              타임라인 (씬 목록)                     │
└─────────────────────────────────────────────────────┘
```

#### 주요 작업

**씬 추가하기**
1. 타임라인에서 ➕ 버튼 클릭
2. 새 씬이 추가됨
3. 씬을 클릭하여 편집

**요소 추가하기**
1. 오른쪽 씬 편집 패널에서 "텍스트", "이미지", "도형" 선택
2. 캔버스에 요소가 추가됨
3. 속성 패널에서 스타일 조정

**배경 변경하기**
1. 씬 편집 패널 → "배경" 탭
2. 단색, 그라데이션, 이미지 중 선택
3. 색상 선택기로 커스터마이징

**씬 순서 변경**
- 타임라인에서 씬을 드래그하여 순서 변경

**재생 시간 조정**
- 씬 편집 패널 → "타이밍" 탭
- 재생 시간 입력 (0.5초 ~ 30초)

### 5단계: 템플릿 사용 (선택사항)

전문가 디자인 템플릿으로 빠르게 시작:

1. 상단 탭에서 "템플릿" 클릭
2. 카테고리별 템플릿 탐색
3. 원하는 템플릿 선택
4. 자동으로 씬과 디자인이 적용됨

### 6단계: 내보내기

완성된 비디오를 내보내기:

1. 우측 상단 "내보내기" 버튼 클릭
2. 포맷 선택 (MP4, GIF, 이미지 시퀀스)
3. 품질 선택 (720p, 1080p, 4K)
4. 다운로드

## 🎨 디자인 팁

### 효과적인 비디오 만들기

**1. 씬 구성**
- 첫 씬: 강력한 후크 (3초 이내)
- 중간 씬: 핵심 메시지 (5-7초)
- 마지막 씬: CTA (2-3초)

**2. 텍스트 사용**
- 큰 텍스트: 주요 메시지 (48-72px)
- 작은 텍스트: 부가 정보 (24-36px)
- 대비 높은 색상 사용

**3. 색상 조합**
- 브랜드 색상 활용
- 그라데이션으로 프리미엄 느낌
- 배경과 텍스트 대비 확보

**4. 플랫폼별 최적화**

| 플랫폼 | 형식 | 길이 | 특징 |
|--------|------|------|------|
| YouTube | 16:9 | 15-60초 | 정보 중심, 전문적 |
| Instagram | 9:16 | 10-30초 | 감성적, 비주얼 중심 |
| TikTok | 9:16 | 7-15초 | 역동적, 트렌디 |
| LinkedIn | 16:9 | 15-45초 | B2B, 데이터 중심 |

## 🛠️ 고급 기능

요소별 애니메이션 설정:

씬 간 전환:
- **Fade**: 부드러운 전환
- **Slide**: 슬라이드 전환
- **Zoom**: 확대/축소 전환
- **Dissolve**: 녹아드는 효과

### 배경 음악 (곧 출시)

- 로열티 프리 음악 라이브러리
- 볼륨 조절 및 페이드 효과
- 자동 배경음악 추천

**Q: 비디오 길이 제한이 있나요?**

A: 네, AI가 생성한 비디오는 자유롭게 편집 가능합니다.

**Q: 템플릿을 수정할 수 있나요?**
A: 네, 템플릿도 완전히 커스터마이징 가능합니다.

**Q: 내 이미지를 추가할 수 있나요?**
A: 네, 이미지 업로드 기능이 곧 추가됩니다.

## 🔄 워크플로우 예시

### 완전 AI 자동화
```
1. 토픽 입력: "신제품 출시"
2. AI 비디오 생성 클릭
3. 자동 생성 완료 (30초 소요)

```

5. 재생 시간 미세 조정
6. 내보내기
⏱️ 총 소요 시간: ~5분
